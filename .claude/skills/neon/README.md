# Neon Database Skills

Future Claude Code skills for working with Neon PostgreSQL.

## Planned Skills

- `new-migration.md` - Create a new Neon database migration
- `new-schema.md` - Create a new Zod schema for a Neon table
- `neon-troubleshoot.md` - Debug Neon connection and query issues

## When to Use Neon Skills

Use these skills when working with custom application data that doesn't belong
in MinistryPlatform:

- Prayer widget data
- Anonymous voting/polls
- Session/cache data
- Analytics/event tracking
- Custom app data that doesn't fit MP's structure
