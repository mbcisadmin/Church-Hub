## Maskable Icons

Maskable icons should have important content within the "safe zone" (80% of the canvas from the center). This ensures the icon looks good when the OS applies different masks (circle, rounded square, etc.).

## Recommended Tools

- **Figma/Sketch**: Design the icons
- **PWA Asset Generator**: Generate all sizes automatically
- **Maskable.app**: Test maskable icons
